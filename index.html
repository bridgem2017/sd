골판박스, 칼라박스, 소량인쇄를 빠르게 선택하고
필요 항목만 메일로 바로 전달하세요

소비자가 선택한 항목이 에스디컴퍼니 메일로 자동 전달되도록 설계된 완성형 박스 주문제작 홈페이지입니다. 자동 견적 금액 노출 대신, 실제 문의에 필요한 선택 항목 중심으로 구성했습니다.

주요 서비스
Service 01

골판박스 주문제작

골판 박스 형태, 색상별 재질, 두께 정보를 보고 바로 견적 문의 페이지로 이동할 수 있습니다.

견적페이지 이동

1. 박스 형태

2. 골판지 색상별 재질

3. 골판지 두께정보

Service 02

칼라박스 주문제작

칼라박스 형태, 종이정보, 코팅/박인쇄/기타 가공 안내를 제공하고 바로 문의로 연결합니다.

견적페이지 이동

1. 박스 형태

2. 칼라박스 종이정보

3. 가공 안내

Service 03

소량인쇄

소량 제작의 장점과 에스디컴퍼니의 소량 인쇄 대응 장점, 그리고 가능 박스 형태를 한 번에 확인할 수 있습니다.

견적페이지 이동

1. 소량인쇄 장점 / 2. 에스디컴퍼니 소량 인쇄 장점

3. 소량인쇄 박스형태

Manufacturing Process

제조공정 안내

골판박스 주문제작과 칼라박스 주문제작의 대표적인 제조공정을 설명과 이미지로 안내합니다.

골판박스 주문제작 공정

원단 입고 → 인쇄 → 톰슨 → 접합 → 검수 → 포장 순으로 진행되며, 골판 두께와 구조에 따라 공정 조건이 달라집니다.

칼라박스 주문제작 공정

인쇄 → 표면가공 → 합지 → 톰슨 → 접착 → 포장 순으로 진행되며, 코팅·박인쇄·후가공 선택에 따라 추가 공정이 반영됩니다.

Inquiry Workflow

선택 항목이 에스디컴퍼니 메일로 전달되는 견적 문의

자동견적금액을 노출하지 않고, 소비자가 선택한 서비스 유형, 박스형태, 재질/지류, 두께/가공, 수량, 인쇄 여부 등 핵심 항목을 정리해 메일로 전달합니다.

  • 골판박스 주문제작 / 칼라박스 주문제작 / 소량인쇄 선택 가능
  • 서비스별 상세 옵션 자동 노출
  • 선택 항목 요약 후 문의 메일 전송
  • 관리자 없이 FormSubmit 방식으로 운영 가능
회원가입 / 로그인

실운영용 회원가입 구조 안내

사이트가입, 카카오톡가입, 네이버가입을 위해서는 각 서비스의 개발자 등록과 인증키 연결이 필요합니다. 이 업로드용 완성파일에는 즉시 배포 가능한 UI와 설정 연결 지점이 포함되어 있습니다.

사이트가입

실운영용으로는 Supabase 또는 Firebase 연결이 필요합니다. 현재 파일에는 연결 위치와 예시 UI가 포함되어 있습니다.

카카오톡가입

카카오 디벨로퍼스에서 JavaScript 키, Redirect URI 설정 후 연결해야 실제 작동합니다.

네이버가입

네이버 개발자센터에서 Client ID, Callback URL 설정 후 연결해야 실제 작동합니다.

중요: 카카오 / 네이버 / 실운영용 사이트가입은 이 파일만으로 즉시 활성화되지 않습니다. 실제 연동에는 앱 등록, Redirect URI, API 키, 개인정보처리방침 페이지가 필요합니다. 이 파일에는 바로 붙일 수 있도록 연결 위치와 UI가 준비되어 있습니다.
